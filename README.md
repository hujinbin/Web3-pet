# Web3-pet
Web3 电子宠物 数字藏品

## 项目介绍

Web3-pet是一个基于区块链技术的电子宠物平台，让用户可以领养、培养和合成独特的数字宠物。

### 核心功能

- **宠物数据上链**：所有宠物的属性、基因和所有权数据均存储在区块链上，确保真实性和永久性
- **免费领养**：用户可以免费领养自己的第一只宠物，无需支付ETH（仅需支付gas费）
- **金币系统**：
  - 平台内置金币经济系统，金币数据同样上链存储
  - 用户可通过每日签到获取金币奖励
  - 金币可用于宠物培养、购买道具等功能
- **宠物合成**：两只宠物可以合成产生全新的宠物，继承父母的部分特征和基因
